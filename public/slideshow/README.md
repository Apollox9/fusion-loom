# Slideshow Images

Add your slideshow images to this folder. The images should be named:
- slide1.jpg
- slide2.jpg
- slide3.jpg
- etc.

The slideshow will automatically display all images in this folder, cycling through them every 4 seconds.

Recommended image dimensions: 1920x1080 (16:9 aspect ratio)
Supported formats: JPG, PNG, WebP
